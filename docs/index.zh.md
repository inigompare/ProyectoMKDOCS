# MkDocs 项目文档

![项目封面](img/portada.png)

## 📖 简介

欢迎来到本项目的 **元文档 (Meta-Documentation)**。本网站一步步记录了在 **Windows 和 PowerShell** 环境下，使用静态站点生成器 (SSG) **创建、配置和部署** 静态网站的完整技术过程。

!!! info "什么是元文档？"
本站点是一个 **递归文档** 的例子：你正在阅读关于如何创建这份文档本身的文档。这是一个展示真实工作流程的实践练习，包括遇到的错误及其解决方案。

## 🎯 项目目标

主要目标是展示对 **ASIR (网络计算机系统管理)** 课程以下技术能力的掌握：

- [x] 开发环境的安装和配置
- [x] 使用版本控制工具 (Git)
- [x] 在云端部署 Web 应用程序
- [x] 技术问题解决 (故障排除)
- [x] 专业技术文档编写

## 🛠️ 使用的技术

| 技术                    | 版本  | 用途                   |
| ----------------------- | ----- | ---------------------- |
| **Python**              | 3.12+ | 运行 MkDocs 的基础语言 |
| **MkDocs**              | 1.6+  | 静态站点生成器         |
| **Material for MkDocs** | 9.5+  | 具有高级组件的专业主题 |
| **Git**                 | 2.43+ | 版本控制               |
| **GitHub Pages**        | -     | 用于部署的免费托管     |
| **PowerShell**          | 7+    | Windows 上的命令终端   |
| **VS Code**             | 1.85+ | 代码编辑器             |
