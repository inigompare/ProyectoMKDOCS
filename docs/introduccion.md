# Introducción

Históricamente, SSH era exclusivo de Unix/Linux. Sin embargo, Microsoft ahora incluye una implementación nativa de **OpenSSH** en Windows.

## ¿Qué vamos a hacer?

1. Activar la característica opcional en Windows.
2. Configurar el servicio para que arranque automáticamente.
3. Asegurar la configuración editando el archivo `sshd_config` en Windows.

## Ventajas

Permite administrar servidores Windows de forma remota mediante línea de comandos segura, sin depender de herramientas gráficas pesadas como Escritorio Remoto (RDP).
